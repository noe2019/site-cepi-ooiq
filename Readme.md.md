# Site CEPI Coach – Préparation Examen OIIQ

Ce site présente les formations, simulations et programmes d’accompagnement pour les candidats à l’examen de l’OIIQ.

## Déploiement

1. Crée un dépôt GitHub nommé `site-cepi-ooiq`.
2. Téléverse les fichiers du dossier présent.
3. Va dans **Settings → Pages → Source → Branch: main / root**.
4. Enregistre → Ton site sera accessible à l’adresse :
   `https://tonpseudo.github.io/site-cepi-ooiq`
